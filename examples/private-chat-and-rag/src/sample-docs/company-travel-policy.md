# 사내 출장 규정

## 1. 적용 범위

본 규정은 전 직원의 국내외 출장에 적용됩니다.

## 2. 국내 출장

### 숙박비 (1박 기준)

| 직급 | 숙박비 상한 |
|------|------------|
| 임원 | 15만원 |
| 팀장 | 12만원 |
| 일반 | 10만원 |

### 교통비

- 고속철도(KTX): 일반석 실비 지급
- 항공: 국내선 이코노미 실비 지급
- 자가용: km당 300원 지급

### 식비

- 1일 3만원 정액 지급 (영수증 불요)

## 3. 해외 출장

- 숙박비: 국가별 상한 기준 적용 (별표 참조)
- 항공: 이코노미 클래스 (12시간 이상 비행 시 비즈니스 가능)
- 일비: 국가별 정액 지급

## 4. 출장 신청

1. 출장 3일 전까지 전자결재 시스템으로 신청
2. 팀장 승인 → 경영지원팀 확인
3. 해외 출장은 본부장 추가 승인 필요

## 5. 정산

- 출장 종료 후 5영업일 이내 정산
- 영수증 원본 첨부 (교통비, 숙박비)
- 법인카드 사용 권장
